## Install

### main

```sh
/bin/bash -c "$(curl -fsSL https://git.io/dot-slime-hatena)"
```

### develop

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Slime-hatena/dotfiles/develop/scripts/clone.sh) develop"
```
